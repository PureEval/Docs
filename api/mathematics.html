sort() ​

将数组按照一定规则排序。

  • Type
  • Details

第一个参数传入一个规则函数，第二个参数传入一个待排序的数组，返回值为排序后的数组。

规则函数可以填写 _ 来代表采取默认规则。

规则函数与 Array.prototype.sort 所使用的规则函数相同。

  • Example
js
sort(under(), [3, 2, 1]); //[1, 2, 3]
