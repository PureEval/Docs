Bind ​

bind() ​

将函数的某些参数提前绑定。

  • Type
  • Details

第一个参数传入一个函数，接下来传入的参数不超过该函数参数个数个参数。

对于每个参数，若传入的是一个确定的值，则将直接绑定在原函数上；若传入的是占位符 _，则表明该位置仍然需要传入参数。

返回值即为对原函数按上述参数绑定值后的新函数。

  • Example
js
const foo = (a, b, c) => a + b + c;
bind(foo, 1, _, 3)(4); //8
const foo = (a, b, c) => a + b + c;
bind(foo, 1, _, 3)(4); //8
