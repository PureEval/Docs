when() #

构造一个参数符合条件后执行一定操作的函数。

  • Type
  • Details

初始传入两个函数类型的参数（记为 ），返回值为传入一个参数的函数。

该函数的行为是：将参数传给 函数，若 函数返回值为 true，则将执行 函数并返回其返回值；相反，如果 函数的返回值为 false，则将原样返回传入的参数。

显然， 函数必须为一个单元函数，而 函数不接收参数。

如果你希望 函数与 函数接受相同的参数，请参见 when_v。

  • Example
js
const foo = when(
  (v) => v == 114515,
  () => "homo!"
);

when(114514); //"homo!";

when_v() #

构造一个参数符合条件后执行一定操作的函数。

  • Type
  • Details

初始传入两个函数类型的参数（记为 ），返回值为传入一个参数的函数。

该函数的行为是：将参数传给 函数，若 函数返回值为 true，则将执行 函数并返回其返回值；相反，如果 函数的返回值为 false，则将原样返回传入的参数。

特别的，传入 函数的参数将与传入 函数的参数相同。

显然， 函数均为单元函数。

如果你希望 不接受参数，请参见 when。

  • Example
js

  (v) => v == "is you!",
  (v) => "homo" + v
);

when("is you!"); //"homo is you!";
